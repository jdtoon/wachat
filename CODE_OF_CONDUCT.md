# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cov] as its code of
conduct. By participating, you agree to abide by its terms.

## Reporting

If you observe behavior that violates the Code of Conduct, please report it
privately to **junaidd0306@gmail.com**. Reports will be handled in confidence.

The Code of Conduct applies to all project spaces: this repository's issues,
pull requests, discussions, and any related communication channels.
